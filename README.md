## Tech Stack

### Frontend
- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router

### Backend
- .NET 10
- Entity Framework Core
- PostgreSQL (SQLite in development)
- JWT Authentication
- BCrypt

### Deployment & DevOps
- Docker Compose
- Nginx
- Terraform (AWS EC2 + Elastic IP)
- GitHub Actions CI/CD
- GitHub Container Registry (`ghcr.io`)
